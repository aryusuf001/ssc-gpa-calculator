# SSC GPA Calculator 

A high-performance open-source JavaScript implementation built to calculate the Secondary School Certificate (SSC) and Dakhil examination GPA accurately. This script perfectly aligns with the official grading guidelines set by the Ministry of Education and the Board of Intermediate and Secondary Education in Bangladesh.

## 🚀 Live Tool
The production-ready, fully interactive web application is available here:
👉 **[SSC GPA Calculator (sscgpacalculator.com)](https://sscgpacalculator.com/)**

---

## ✨ Features
* **Precise Grade Point Mapping:** Instantly maps individual subject marks or letter grades to the standard Bangladeshi 0.00 to 5.00 scale.
* **Automated 4th Subject Rule:** Properly processes the optional (fourth) subject bonus logic—subtracting 2.00 points from the earned optional Grade Point (GP) and pooling the remaining surplus points into the core total.
* **Instant Fail Detection:** Instantly flags an overall GPA of 0.00 if a student receives an F grade in any mandatory core subject, ensuring strict compliance with board policy.
* **Lightweight Architecture:** Written in pure, dependency-free JavaScript for blazing-fast execution.
